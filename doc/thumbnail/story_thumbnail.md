StoryThumbnail
====
리스트 화면에서 하나의 스토리 썸네일이 어떻게 표시되어야 할지 지정합니다.

```json
{
    "title" : "TITLE",
    "description" : "DESCRIPTION",
    
    "images" : 5
}
```

Properties
----
* __title__ : 리스트 화면에 보여질 타이틀 값입니다.
* __description__ : 타이틀 아래에 보여질 간략한 설명을 지정합니다.
* __images__ : 썸네일 이미지의 갯수를 지정합니다. 썸네일 이미지들은 스크롤될 수 있습니다.
