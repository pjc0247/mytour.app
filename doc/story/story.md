Story
====
상세보기 화면에서의 전체적인 스타일을 지정합니다.

```json
{
    "title" : "TITLE",
    
    "background" : "black",
    "foreground" : "white"
}
```

Properties
----
* __title__ : 상단 네비게이션 바에 표시될 타이틀 값입니다.
* background : 백그라운드 색상입니다. 지정하지 않으면 하얀색으로 표시됩니다.
* foreground : 포그라운드 글자 색상입니다. 지정하지 않으면 검은색으로 표시됩니다.
